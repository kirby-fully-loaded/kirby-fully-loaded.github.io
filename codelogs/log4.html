CSS uses color in the form of code, and applies the color defined by the code to a specific element. Color is a property of an element that must be defined through CSS, either in-line or on a CSS Styles sheet. If color is not defined, it will use the browser's default settings; these are usually black for foreground elements, and white for background elements.

Color can be applied to an element in the following ways:

    • This defines the color of foreground objects like text.
    • This defines the background color of an element; it can be useful for distinguishing parts of a page's content, or improving readability of a webpage.
    • This defines the color of an element's border, if present. Like it is useful for distinguishing part of page content or improving readbility.
