CSS Color

CSS uses color in the form of code, and applies the color defined by the code to a specific element. Color is a property of an element that must be defined through CSS, either inline, internally, or on a CSS Styles sheet. If color is not defined, it will use the browser's default settings; these are usually black for foreground elements, and white for background elements.

Color can be applied to an element in the following ways:

  • color:
    • This defines the color of foreground objects like text.
  • background-color:
    • This defines the background color of an element; it can be useful for distinguishing parts of a page's content, or improving readability of a webpage.
  • border-color:
    • This defines the color of an element's border, if present. Like background-color: it is useful for distinguishing part of page content or improving readability.

Colors for each of these properties can be defined in different ways through CSS, each of which have their pros and cons.

  • Named
    • background-color: red
    • This is an example of the named color "red." CSS includes a number of preset, named colors to choose from. Unless you have specific colors you want to use, named colors are the most likely to display your content as-intended on a given display, because they are most likely to be supported by a given browser. However, the preset nature of these colors is necessarily limitng, so one of the other means of defining color values might better serve your needs.
  • RGB
    • background-color: rgb(0,255,0)
    • background-color: rgb(0%,100%,0%)
    • These two color fields both use RGB to determine their display, but use different scales to determine their values. RGB uses three different values for Red, Green, and Blue (hence the name) each of which scale individually. The two scales are 0-255, as used in the first color field, and 0%-100% as used in the second field. In both cases, the values determine the intensity of the corresponding color; the percentage scale measures percentage of each color's intensity, not the percentage of its representation relative to the other colors in the final displayed color. Because of its capacity for percentage-based color manipulation, RGB may be the most accessible format for non-coders and novice coders wishing to create custom color values for their websites and pages.
  • Hexadecimal
    • background-color: #0000ff
    • This is an example of the color blue defined using hexadecimal code. In hexadecimal, colors are determined by assigning different values of red, green, and blue, following the format #RRGGBB, in values ranging from 00 to ff. Because the red and green values are each set to 00, the lowest value, they do not render. In contrast, the blue value is set to ff, the highest value, so it appears with the greatest intensity.
